# Task History Archives

This folder stores long-form task ledgers that used to live in repo-root
`TASK.md`.

The repository no longer uses a root `TASK.md` handoff file. Active work should
start from `AGENTS.md`, relevant specs under `openspec/specs/`, and active
change artifacts under `openspec/changes/` when the task is substantial.

## Archives

| File | Scope |
|---|---|
| `sync-refactor-task-history.md` | Verbatim pre-split `TASK.md` history for sync-refactor Phases 2.7 through OpenSpec migration work. |
